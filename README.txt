WEB PROJECT 02
Important information

This website runs with MongoDB.
The server is listening on port 5000. 

Please execute the following command on the console, while on the correct path:
npm run devStart

There are no posts by default. Please create a post by clicking on the "New Post" button on the Index.
You can create posts, edit the posts, and delete the posts.
Please add a cover image if you feel like it. If you don't, a generic image will be displayed by default as the cover.
